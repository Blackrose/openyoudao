openyoudao

youdao for linux 有道linux客户端

Introduction

openyoudao是一个有道字典在linux下的查字客户端，在取词翻译的基础上，对查询到的信息进行有效的整合，然后开发出更多、更有用的功能。

目前openyoudao已经实现了屏幕取词翻译功能，但是跟一个完善的应用程序还有很长的一段距离。

欢迎使用openyoudao并反应程序存在的Bug，我们会在您的支持下努力完善程序，并开发出更多有价值的功能。

Features

1. 取词稳、准、快

2. 界面没有按钮
